# Recetas App

Aplicación móvil de recetas de cocina desarrollada con **React Native** y **Expo**, diseñada para ofrecer una experiencia intuitiva y completa en la búsqueda, exploración y gestión de recetas favoritas.

---

#  Definición del Problema

En la actualidad, muchas personas enfrentan dificultades para encontrar inspiración culinaria y organizar sus recetas favoritas. Los principales problemas son:

* **Dispersión de información:** las recetas están distribuidas en múltiples fuentes como libros, blogs, redes sociales y videos.
* **Falta de organización:** no existe un lugar centralizado para guardar y clasificar recetas.
* **Dificultad para encontrar recetas rápidas:** los usuarios necesitan filtrar por tiempo, dificultad o categoría.
* **Sin persistencia de favoritos:** al encontrar una receta que les gusta, no hay una forma sencilla de guardarla para consultarla posteriormente.

**Recetas App** soluciona estos problemas ofreciendo un catálogo organizado de recetas con funcionalidades de búsqueda, filtros, favoritos y una interfaz limpia e intuitiva.

---

#  Justificación del Alcance

La aplicación permite a los usuarios:

* Explorar un catálogo de recetas organizadas por categorías.
* Buscar recetas por nombre, categoría o ingredientes.
* Filtrar recetas por categoría para encontrar rápidamente lo que buscan.
* Visualizar el detalle completo de cada receta (ingredientes, pasos, tiempo y dificultad).
* Guardar recetas favoritas mediante persistencia local utilizando **AsyncStorage**.
* Acceder rápidamente a sus recetas favoritas desde una sección dedicada.
* Navegar de forma intuitiva con transiciones fluidas entre pantallas.

#  Requisitos Previos

Antes de ejecutar la aplicación, asegúrate de tener instalado:

| Herramienta | Versión recomendada           | Comando para verificar |
| ----------- | ----------------------------- | ---------------------- |
| Node.js     | v18.x o v20.x (LTS)           | `node --version`       |
| npm         | v9.x o superior               | `npm --version`        |
| Expo CLI    | Opcional (se utiliza con npx) | `npx expo --version`   |

---

#  Instalación y Ejecución

## 1. Clonar el repositorio

```bash
git clone https://github.com/thomasx4/recetas-app.git
cd recetas-app
```

## 2. Instalar las dependencias

```bash
npm install
```

## 3. Ejecutar la aplicación

```bash
npx expo start
```

Una vez iniciado el servidor de desarrollo, podrás:

* Presionar **a** para abrir la aplicación en Android.
* Presionar **w** para abrir la versión web.
* Escanear el código QR con la aplicación **Expo Go** en tu dispositivo móvil.
