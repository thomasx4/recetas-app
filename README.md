# Recetas App

Aplicación móvil de recetas de cocina desarrollada con React Native y Expo, diseñada para ofrecer una experiencia intuitiva y completa en la búsqueda, exploración y gestión de recetas favoritas.

---

## Definición del Problema

En la actualidad, muchas personas enfrentan dificultades para encontrar inspiración culinaria y organizar sus recetas favoritas. Los problemas principales son:

- **Dispersión de información**: Las recetas están distribuidas en múltiples fuentes (libros, blogs, redes sociales, videos).
- **Falta de organización**: No existe un lugar centralizado para guardar y clasificar recetas.
- **Dificultad para encontrar recetas rápidas**: Los usuarios necesitan filtrar por tiempo, dificultad o categoría.
- **Sin persistencia de favoritos**: Al encontrar una receta que les gusta, no hay forma de guardarla para consultarla después.

**Recetas App** soluciona estos problemas ofreciendo un catálogo organizado de recetas con funcionalidades de búsqueda, filtros, favoritos y una interfaz limpia e intuitiva.

---

## Justificación del Alcance

La aplicación permite a los usuarios:

- **Explorar** un catálogo de recetas organizadas por categorías
- **Buscar** recetas por nombre, categoría o ingredientes
- **Filtrar** por categoría para encontrar rápidamente lo que buscan
- **Ver detalle completo** de cada receta (ingredientes, pasos, tiempo, dificultad)
- **Guardar recetas favoritas** con persistencia local (AsyncStorage)
- **Acceder rápidamente** a sus recetas favoritas desde la pestaña dedicada
- **Navegación intuitiva** con transiciones fluidas entre pantallas
